File management is more flexible in this build:

- Copy, Rename, and Delete can now work with all file types, not only audio and video.
- Added a file-name filter so you can target groups of files by shared text, such as a name prefix plus index.
- Delete now double-checks that the preview still matches the current filters before moving anything to Trash.
- Rename patterns now support `{date}` and show a small in-window guide for available variables.
- Large filtered previews are lighter: the app shows a capped sample while keeping accurate totals and applying the full filtered list when you run an action.
